# Underground Hold'em

싱글 페이지로 실행되는 텍사스 홀덤(1인 + 봇 3인) 게임입니다.
프로미넌스 포커의 분위기(다크 카지노/펠트 테이블/UI 톤)를 참고한 오리지널 구현입니다.

## 실행 방법

1. 이 폴더에서 `index.html`을 브라우저로 엽니다.
2. 우측 상단 `Next Hand` 버튼으로 핸드를 시작합니다.
3. 브라우저 WebGL이 켜져 있으면 자동으로 3D 모드가 활성화됩니다.

## 포함된 기능

- 노리밋 텍사스 홀덤 라운드
- 블라인드/딜러 버튼 순환
- SB/BB 좌석 표시 및 상단 SB/BB 정보 표시
- 프리플랍/플랍/턴/리버/쇼다운
- 폴드, 체크/콜, 레이즈(슬라이더), 올인
- 홀카드 `Peek Cards` 버튼(누르고 있는 동안만 카드 확인)
- 딜러 카드 투척 애니메이션(홀카드/커뮤니티 카드)
- Three.js + glTF(`Soldier.glb`) 기반 실시간 3D 테이블/조명/인물 렌더링
- 턴/올인/쇼다운 카메라 시네마틱 연출(자동 포커스/줌/셰이크)
- 카드 앞면 텍스처(랭크/슈트) 3D 렌더링
- 7장 기준 핸드 랭크 판정
- 기본 봇 의사결정 로직
- 옆 시점 테이블, 딜러/플레이어 사람형 아바타, 카지노 소품, 올인 스탠드업 모션
- 데스크톱/모바일 반응형 UI

## 참고

- 완전 복제(원본 게임의 리소스/브랜드/에셋 재사용)가 아니라, 유사 무드의 신규 UI로 제작되었습니다.
- `vendor/three`에 `three.min.js`, `GLTFLoader.js`, `SkeletonUtils.js`, `Soldier.glb`를 로컬 포함해 오프라인에서도 실행됩니다.
